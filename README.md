# Горячие клавиши
#### Выделить все:
Ctrl+A 
#### Копировать:
Ctrl+C
#### Вставить:
Ctrl+V
#### Сохранить: 
Ctrl+S
#### верезать:
Ctrl+X
#### Отменить:
Ctrl+Z
# Работа с git
### Скачать проект:
```bash
git clone <URL>
```
### Учесть все файлы: 
```bash
git add .
```
### Сохранить  комммит:
```bash
git commit -m "..." 
```
### Отправка в облако:
```bash
git push
```

### Принятие из облака:
```bash
git pull
```
